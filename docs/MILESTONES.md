# CivicComms Milestones

## Milestone 0-7: v0.1.0 Foundation

Acceptance criteria:

- FastAPI app imports and serves `/`, `/health`, and `/civiccomms`.
- Source-review, meeting-summary, ordinance-summary, newsletter, FAQ, and audience-variant APIs return deterministic support objects.
- Docs clearly state no autonomous publication, no campaign/advocacy content, no legal advice, no certified translation, no live LLM calls, no social media posting, and no communications system-of-record behavior.
- Browser QA passes at desktop and mobile widths.
- `scripts/verify-release.sh` passes.
